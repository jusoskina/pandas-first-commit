# Verify pandas contribution

Follow `pandas-verify-contribution` (`.cursor/skills/pandas-verify-contribution/SKILL.md`).

Run or recommend checks. Always output the **Checks run report** table with pytest, pre-commit, and checker rows. Then output the **PR readiness check**.

On stage 7 success (no required `FAIL`), immediately follow `pandas-ship-contribution` in the same response. Do not open a PR or push unless the user explicitly asks.
